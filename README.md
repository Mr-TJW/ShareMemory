# 消息处理
* 采用数据队列处理方式
* 接收到一帧数据里面包含多个 dlt698 报文，依次处理
* 一组 dlt698 报文被分成多个数据帧，等接收到完整 dlt698 报文后再进行处理

## 通信方式
* TCP 客户端模式
* 收发数据均采用数据队列方式，本次接收的数据不会覆盖上一次还未处理的数据
